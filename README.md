# amma-csi-pain-prediction
WiFi CSI based non-intrusive pain prediction system
## First Research Finding - March 25, 2026

Amma reported leg pain between 6:00-6:30 AM.
WiFi signal showed 36 dBm deviation starting at 06:00:05.
First confirmed pain correlation detected.
System detected movement before conscious pain registered
Amma suffered silently for 5-10 minutes without calling for help.
Automated heat activation could have provided relief before peak pain.
